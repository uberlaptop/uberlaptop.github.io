Convert any audio or video file from one format to another using FFmpeg

I’ll start with a small introduction, for people working with audio and video files on opensource platform, FFmpeg is a command line swiss army knife. It’s a multi-purpose utility tool, but most of the time I use it to convert files from one format to another. One more thing, the ‘FF’ stands for Fast Forward.

Starting with installation:

Make sure you have the root access, download the latest version of ffmpeg, currently the latest version of ffmpeg at the time of writing this article is 2.2.

$ cd /tmp

$ wget -c http://ffmpeg.org/releases/ffmpeg-2.2.tar.gz

Once the download is done, extract the files and get inside the ffmpeg directory.

$ tar -xvzf ffmpeg-2.2.tar.gz

$ cd ffmpeg-2.2

Now we start with the installation. Execute the below mentioned commands one after another.

First we’ll install some necessary packages required by ffmpeg.

$ sudo apt-get update

$ sudo apt-get install yasm libmp3lame0 libmp3lame-dev libavcodec-extra-53 libx264-dev x264

Then configure and build ffmpeg.

$ sudo ./configure —enable-libmp3lame ; sudo make ; sudo make install

This will take some time, so have patience.

Once the installation is complete, one final step need to be taken, we’ll need to create a symlink.

$ sudo ln -sf /usr/local/bin/ffmpeg /usr/bin/ffmpeg

And we are done with the installation.

Converting File Formats:

To convert a flash video file .flv to a .mp4 format:

$ ffmpeg -i samplevideo.flv -strict -2 -qscale 3 samplevideo.mp4

If you only need a sound file, you may want to convert a mp4 file to mp3.

$ ffmpeg -i samplevideo.mp4 sampleaudio.mp3

Or

$ ffmpeg -i samplevideo.mp4 -b:a 192K -vn sampleaudio.mp3

To convert .avi file to .mp4 format:

$ ffmpeg -i samplevideo.avi -acodec libfaac -b:a 128k -vcodec mpeg4 -b:v 1200k -flags +aic+mv4 samplevideo.mp4

To convert .mp4 to .mpg

$ ffmpeg -i samplevideo.mp4 samplevideo.mpg

To convert a group of images into a video sequence:

$ ffmpeg -f image2 -i image%d.jpg testvideo.mpg

To generate multiple images out of a video:

$ ffmpeg -i samplevideo.mpg image%d.jpg

You may require this, to decide on a thumbnail for your video.

That’s mostly all I use ffmpeg for.

If you are not much of a command line person, then you may want to try WinFF, its very simple and easy to use and works equally well.
