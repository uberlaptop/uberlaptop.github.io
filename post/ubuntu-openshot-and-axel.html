Ubuntu, OpenShot and Axel

Being an opensource enthusiast and trying to do multimedia stuff is not easy. When you compare with the kind of tools and applications available on other operating systems like Mac and Windows, opensource applications don’t seem to be capable enough.

I have been a ubuntu user for past 7 years, but the introduction of unity didn’t feel good, it was a very heavy memory consumer. So I did hold on to ubuntu 10.10 for quiet a long time and when I made the switch, I switched to Ubuntu 12.10 but not the one with unity, I switched to ubuntu studio 12.10.

Ubuntu studio has xfce4 as the window manager, which is quite simple and even better than unity.

For opensource enthusiasts interested in designing, there’s a desktop application gimp which has all the tools and features of photoshop. If you would like to learn gimp in further details, do checkout LinuxDesigner. I am not much of a designer, but whenever i need to create a little bit of titles, video credits or do some minor image changes, I use an online tool Pixlr.

For those interested in creating movies or youtube videos, there are quite a good number of applications available. If you are looking for something with lots of abilities to various kinds vfx, then go for Blender or Kdenlive, apart from these, other really good tools to try out are avidemux, cinelerra, pitivi and openshot.

Openshot is the most simple and easy to use video editor I have worked with, I still use it, and if you are not much of a vfx guy, I would recommend you to give it a try.

Let me know if there any other online or opensource tools and applications you use for your multimedia projects.

One more recommendation, If you are a heavy downloader, then, try out Axel, it’s a command line tool to speed up your downloads.

$ sudo apt-get install axel

To download a file:

$ axel [ file to download ]
